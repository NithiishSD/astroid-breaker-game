## 🕹️ About the Game  
Asteroid Breaker is a **retro-style 2D brick breaker game** where you:  
- Control a paddle to bounce the ball 🏓  
- Break all asteroid blocks 🧱  
- Survive with accurate timing & precision 🎯  
- Enjoy classic arcade-style gameplay 🎶
